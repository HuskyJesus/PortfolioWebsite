What Do They See?

FIEA Application Project  ·  2025  ·  Solo Developer  ·  Unity / C#  ·  VR

A short first-person VR experience built as part of the FIEA graduate application. Players put on a pair of glasses and swap between three colored lenses - each one revealing a different layer of a hidden world that exists within the same scene. The core mechanic uses custom shaders to selectively show and hide objects depending on which lens is active, creating a puzzle experience built around perception and discovery.

How It Works

All objects in the experience exist in the scene simultaneously - custom shaders filter what is visible based on the active lens color. Swapping lenses is a physical in-world action: the player reaches up and changes the lens on the glasses they are wearing, revealing a new layer of the environment without loading a new scene.

Experience Walkthrough

  • Red lenses - The player puts on the glasses and sees a code written in the environment that is invisible without the lens.
  • Teleport to the desk - Using the revealed information, the player moves to the next area.
  • Green lenses - Swapping to green reveals a book in the world. Picking it up moves a hidden wall, exposing a safe behind it.
  • Red lenses again - The player uses the code they memorized to unlock the now-visible safe.
  • Purple lenses - The player puts on the final lens, turns around, and is met with a jumpscare.

What I Built

  • Custom shader system that selectively renders objects based on active lens color
  • VR lens-swapping interaction built for headset hand tracking
  • Environmental object interactions - book pickup, hidden wall reveal, safe unlock
  • Teleportation locomotion system
  • Scripted jumpscare sequence with audio and environmental triggers
  • Full level design and environment layout
