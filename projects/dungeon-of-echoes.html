Dungeon of Echoes

Solo Project  ·  Unity / C#  ·  2D Roguelike

A 2D roguelike dungeon crawler built from scratch in Unity using C#. Every system - from player combat to procedural dungeon generation - was designed and implemented solo, with a focus on satisfying moment-to-moment gameplay and replayability through item-driven builds.

What I Built

  • Player combat systems
  • Enemy AI and boss AI
  • Procedural dungeon generation
  • Item-driven stat system
  • Projectile system
